# FHEVM SDK Templates

This directory contains ready-to-use templates for different frameworks.

## Available Templates

### Next.js Template
Location: `../examples/nextjs/`

Full-featured Next.js 14 application demonstrating:
- App Router with React Server Components
- FHEVM SDK integration
- Complete FHE workflow (encrypt, decrypt, compute)
- Multiple example use cases
- API routes for server-side operations

**Quick Start:**
```bash
cd ../examples/nextjs
npm install
npm run dev
```

### React Template
Location: `../examples/react/`

React + Vite template with:
- FHEVM SDK hooks
- Encryption demo
- Lightweight and fast development

**Quick Start:**
```bash
cd ../examples/react
npm install
npm run dev
```

### Node.js Template
Location: `../examples/nodejs/`

Backend service template featuring:
- Server-side FHE operations
- Batch encryption
- API-ready structure

**Quick Start:**
```bash
cd ../examples/nodejs
npm install
npm start
```
